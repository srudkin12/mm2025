The Index of Multiple Deprivation for England and Wales 2025 is detailed in the statistical release at https://www.gov.uk/government/statistics/english-indices-of-deprivation-2025/english-indices-of-deprivation-2025-statistical-release

The first half of the workshop considers artificial data. The artificial data is available in the Session1Python.ipynb file. Slides are mm2025P1.pdf

The second half of the workshop is based on the Index of Multiple Deprivation. There is now code available in both Python and R. The relevant files are Session2Python.ipynb and Session2R.ipynb. The data and shapefiles used are in the GitHub.

Accompanying the code is a commentary, Methods at Manchester Session 1.pdf.
